ToDo List — React
Projeto simples que fiz para aprender React na prática.
A ideia aqui não foi criar nada complexo. O objetivo foi entender como o React realmente funciona no dia a dia: componentes, estado, eventos e manipulação de listas. Coisa básica, mas essencial para qualquer aplicação.
Se você está começando com React, esse tipo de projeto é o equivalente a fazer flexões antes de levantar peso. Simples, direto e necessário.

O que foi praticado:

Estrutura de um projeto React usando Vite
Criação e organização de componentes
Uso de useState para gerenciar estado
Manipulação de listas de tarefas
Eventos de adicionar e remover itens
Separação de estilos usando CSS Modules

Rodando o projeto:

Clone o repositório:

git clone https://github.com/seu-usuario/seu-repositorio.git

Entre na pasta:

cd seu-repositorio

Instale as dependências:

npm install

Execute o projeto:

npm run dev

O Vite vai subir o servidor local e mostrar o endereço no terminal. Normalmente:
http://localhost:5173
Abre no navegador e pronto.
